# Bank Loan Default Risk Analysis

## Overview
Analyzed 49,278 loan records to identify key default 
risk drivers using Python, SQL and Power BI.

## Tools
Python, Pandas, SQLite, Matplotlib, Seaborn, Power BI

## Key Findings
1. Loan purpose P2 has highest default rate at 33%
2. LTV above 90% shows 35% default rate — highest risk segment
3. Low income borrowers default at 43% — nearly double average
4. Overall default rate is 24.39% — 1 in 4 loans defaulting
5. Credit score alone is not a reliable default predictor

## Recommendation
Implement composite risk scoring using LTV ratio, 
income segment and loan purpose before approval.
